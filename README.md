# 🔍 GitHub Finder

Aplicação que consome a API oficial do GitHub para buscar perfis de desenvolvedores em tempo real.

### 🛠 Tecnologias e Conceitos:
- **JavaScript (ES6+):** Uso de `Async/Await` e `Fetch API`.
- **Manipulação de DOM:** Atualização dinâmica da interface.
- **Tratamento de Erros:** Lógica para usuários não encontrados.
